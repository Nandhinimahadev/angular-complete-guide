# Angular Complete Guide

A complete hands-on Angular learning project covering **fundamentals**, **core concepts**, and **real-world examples** — organized into separate apps for clarity.

---

##  Project Overview

| Project | Description |
|----------|-------------|
| **first-app** | Basic Angular fundamentals — components, templates, data binding, and event handling. |
| **core-examples** | Demonstrates core Angular features — parent-child component interaction, custom directives, and pipes. |

---

##  Getting Started

###  Prerequisites
Ensure you have the following installed:
- **Node.js** ≥ 18  
- **Angular CLI** ≥ 17  

Install dependencies:
```bash
npm install

Note: All app run inside angular-workspace
cd angular-workspace


1️.First App
   Location: projects/first-app
   Description : This app demonstrates the basics of Angular — components, data binding, and interaction through events.
   Run cmd:   ng serve first-app

2.Core Examples
   Location: projects/core-examples
   Description : This app demonstrates essential Angular concepts — Parent–Child Component Interaction,Custom Directive,Custom Pipe,Service
   Run cmd:   ng serve core-examples

3.Form Examples
   Location: projects/forms-examples
   Description : This app demonstrates Angular forms — both Template-driven and Reactive approaches with validation and submission handling.
   Run cmd:   ng serve forms-examples
